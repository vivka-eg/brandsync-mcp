# User Management

**Type:** UI Pattern (full screen)
**Category:** data-display, admin
**Responsive:** Desktop, Tablet, Mobile

## Use Case

List, search, filter, and manage a collection of users or records. Use for any admin screen that requires CRUD operations on a list of entities — users, agents, clients, assets, or records. Core pattern in back-office and enterprise applications.

## Components Used

- Table
- Pagination
- Input Fields
- Buttons
- Chips
- Checkbox
- Menu
- Navigation Drawer
- Avatar
- Tabs

## Layout

App shell with collapsible sidebar, toolbar (search + filter + action buttons), sortable data table with pagination.
