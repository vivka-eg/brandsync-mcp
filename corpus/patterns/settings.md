# Settings Page

**Type:** UI Pattern (full screen)
**Category:** settings, admin, form
**Responsive:** Desktop, Tablet, Mobile

## Use Case

Application or account settings screen. Two-column layout: left settings category nav + right form content area. Covers personal profile, notification preferences, security, billing, and organisation sections. Use for any settings or preferences screen in back-office and enterprise applications.

## Components Used

- Navigation Drawer
- Tabs
- Input Fields
- Select
- Switch
- Buttons
- Avatar
- Card
- Badge
