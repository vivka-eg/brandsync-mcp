# Form (Order / Work Order)

**Type:** UI Pattern (full screen)
**Category:** form
**Responsive:** Desktop, Tablet, Mobile

## Use Case

Multi-section data entry form for structured workflows. Use when a user must fill in information across grouped sections to complete an order, request, or work item. Common in field service, logistics, energy, and operations products.

## Components Used

- Input Fields
- Radio Button
- Buttons
- Breadcrumb
- Progress Stepper
- Badge
- Card
- Accordion
- Navigation Drawer
- Avatar
- Menu

## Layout
